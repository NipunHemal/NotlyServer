# **App Name**: CognitoNotes AI

## Core Features:

- Intelligent Document Management: Create, organize, and manage notes using hierarchical groups. Features include detailed DocCards (Title, Preview, Tags, Author, AI indicator), drag-and-drop organization, and a robust markdown editor with block-based editing and inline formatting. DocMetaPanel supports categories and tag input.
- AI-Powered Writing Assistant: Utilize an AI Assist Panel within the editor to suggest relevant tags based on content, predict document categories, and generate summaries for long notes, enhancing content creation and organization. These suggestions appear inline as a helpful tool.
- Secure Collaboration & Version Control: Share documents with defined roles (Owner, Editor, Viewer), manage collaborators, and track viewers. Maintain a Git-style version history with a timeline, allowing users to preview and restore previous versions of notes.
- Personalized SaaS Dashboard & Global Search: Access a central command center featuring a welcome header, analytics stats (notes, shared, favorites), latest groups and documents, and recent activity. A universal search bar allows comprehensive searching by title, tags, content, groups, and collaborators with results in a dropdown overlay.
- Unified Workspace Navigation: Navigate the application with a responsive SaaS dashboard layout, including a collapsible and animated glassmorphism-style sidebar (Home, All Docs, Groups, Recent, Shared With Me, Favorites, Activities, Bin, Settings) and a top header bar. An optional right sidebar provides contextual information.
- Comprehensive Universal Actions: Perform actions on any document or group through a consistent dropdown UI, including Lock, Share, Download, Move, Copy, Duplicate, Make Version, Archive, Add Favorite, Reminder, and soft delete to the Bin with restore options.

## Style Guidelines:

- Primary interactive elements will use a sophisticated muted indigo (`#7070C2`). This provides a professional and calm focal point against the dark theme, evoking focus and depth appropriate for knowledge management.
- The background will be a deep, dark blue-grey (`#16161C`), subtly tinted with the primary hue for a cohesive, minimalist base, reinforcing a high-readability and distraction-free environment.
- An accent color of bright sky blue (`#A3D9F7`) will be used for highlighting actions and providing essential contrast, offering a modern and fresh feel crucial for intuitive interactions on a dark interface.
- AI-specific UI elements will feature a distinct light vibrant cyan (`#7DE8E8`), aligning with the user's request and serving as a clear, energetic marker for AI interaction.
- The primary typeface will be 'Inter' (sans-serif), chosen for its modern, clean, and highly readable qualities, suitable for both headlines and body text, in line with modern SaaS aesthetics.
- Leverage 'Lucide Icons' for all iconography, ensuring a consistent, crisp, and minimalist visual style across the application. Icons will maintain clarity on dark surfaces and support glassmorphism-inspired elements.
- The interface will feature a clean SaaS dashboard layout with large whitespace and card-based components displaying soft shadows and subtle hover elevation. Key architectural elements include a glassmorphism sidebar, top header, main content, and optional right context panel, emphasizing a minimalist yet functional structure.
- Subtle and smooth micro-interactions will be implemented using Framer Motion. This includes elegant card hover effects, fluid sidebar collapse/expand transitions, page transitions, responsive floating button animations, and smooth dropdown opening/closing sequences for a premium feel.
